# Local Life

## Description
This is an application to submit and checkout posts related to local area zipcodes.
Log in to leave a recommendation, or just search zipcodes to see what locals suggest

## Built With
* Express
* Handlebars
* HTML
* CSS
* JavaScript
* NPM
* Express Session
* MySql
